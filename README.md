# two team counter

a java based program for keeping score between two teams
design is based on gnome calculator 

## how to use

download the repository and run the twoTeamCounter.jar file

j = +1 for team 1

n = -1 for team 1

k = +1 for team 2

m = -1 for team 2

## known issues

won't work unless you put in names for both teams
